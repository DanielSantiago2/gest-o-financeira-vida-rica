# 💰 Vida Rica - Gestão Financeira Inteligente

O **Vida Rica** é um aplicativo de gestão financeira moderna, focado em simplicidade e eficiência para o controle de gastos pessoais ou compartilhados (casais). O projeto faz parte do ecossistema de soluções da **Leinad System**.

## 🚀 Funcionalidades

- **Autenticação Segura:** Sistema de login e cadastro integrado ao Firebase Auth.
- **Gestão de Transações (CRUD):** Adicione, visualize, edite e exclua ganhos e despesas em tempo real.
- **Gráficos Interativos:** Visualização estilo "Doughnut" (rosca) via Chart.js para análise rápida de saúde financeira.
- **Controle de Metas Mensais:** Definição de limites de gastos com barra de progresso dinâmica.
- **Notificações Inteligentes:** Sistema de alertas (SweetAlert2) que avisa quando a meta está próxima do limite (90%) ou foi estourada.
- **Categorização Automática:** Filtros inteligentes para organizar moradia, alimentação, lazer, investimentos e mais.
- **Persistência em Nuvem:** Todos os dados são salvos de forma segura no Google Firebase Firestore.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5, CSS3 (Design Responsivo/Dark Mode)
* **Lógica:** JavaScript ES6+ (Módulos)
* **Banco de Dados:** Firebase Firestore
* **Autenticação:** Firebase Auth
* **Gráficos:** [Chart.js](https://www.chartjs.org/)
* **Alertas:** [SweetAlert2](https://sweetalert2.github.io/)
